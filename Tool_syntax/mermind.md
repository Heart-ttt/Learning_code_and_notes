| 元素 | 语法 | 说明 |
|------|------|------|
| 节点 | `A[矩形]`<br>`B((圆形))`<br>`C{菱形}` | `[ ]` 矩形（过程）<br>`(( ))` 圆形（开始/结束）<br>`{ }` 菱形（判断） |
| 连接线 | `A --> B` | 实线箭头 |
|          | `A --- B` | 无箭头线 |
|          | `A -.-> B` | 虚线箭头 |
| 标签 | `A -->|标签| B` | 在线上加文字 |

```mermaid
graph LR
    A[开始] --> B{输入 x}
    B -->|x > 0| C[输出正数]
    B -->|x <= 0| D[输出非正数]
    C --> E[结束]
    D --> E
```

换行：在节点文本中用`<br>`换行
